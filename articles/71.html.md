# 概要
このアプリケーションは, 城好きに向けた城巡りの計画をサポートするためのWebアプリケーションです。
このアプリケーションはマップ上で城の位置を示すことで、簡単に城の位置を知られます

# 技術スタック
- フロントエンド
    - Next.js (App Router)
    - leaflet
    - sass
    - Cloudflare Pages
- バックエンド
    - Hono
    - drizzle-orm
    - wrangler
    - Cloudflare Workers
    - Cloudflare D1
